# Churn Prediction Project

This repository contains datasets and a churn modeling notebook for customer churn analysis.

## Structure

- `data/` - raw CSV datasets used for churn modeling and feature analysis.
- `part3/` - final churn model notebook, error analysis, metrics, and model documentation.

## Key files

- `part3/churn_model.ipynb` - main notebook for building and evaluating the churn model.
- `part3/error_analysis.md` - review of false positives and false negatives.
- `part3/model_card.md` - model summary and evaluation details.
- `part3/metrics.json` - model performance metrics.
- `data/` - dataset sources including customer, orders, web events, support tickets, and intervention history.

## Notes

This project is designed for customer churn analysis and model evaluation. The notebook and supporting files document the modeling workflow, performance insights, and error cases.
